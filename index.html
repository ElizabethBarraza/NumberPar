Número Par o Impar

Este programa permite que el usuario ingrese un número y el sistema determine si es par o impar usando el residuo de la división.

Entrada

Un número entero

Regla

n % 2 == 0

Salida

Par / Impar

¿Cómo funciona?

Si el residuo de dividir el número entre 2 es 0, el número es par. Si el residuo es distinto de 0, el número es impar. 

# Ejemplo (idea principal)
n = int(input("Ingresa un número: "))
if n % 2 == 0:
    print("El número es par")
else:
    print("El número es impar")
Tip: “%” se llama operador módulo (residuo). Hecho para práctica de lógica
